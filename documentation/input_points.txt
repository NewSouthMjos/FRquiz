---------------------ТОЧКИ ДОСТУПА API--------------------
==========================================================
---------------------Для администратора:------------------
Опросники:
1. /api/quiz/new POST:
Создаёт опрос. Обязательные параметры:
name, description.
Необязательные параметры:
start_date, end_date

Пример JSON-данных для использования:
{
    "name": "Опрос удовлетворенности",
    "description": "Общие вопросы удовлетворенности",
}

2.1 /api/quiz/<id>/edit GET:
Возвращает опрос с номером <id>, без списка относящихся
к нему вопросов. 
Обязательные параметры: нет
Необязательные параметры: нет

2.2 /api/quiz/<id>/edit PATCH/PUT:
Обновляет и возвращает опрос с номером <id>, 
без списка относящихся к нему вопросов.
Обязательные параметры: нет.
Необязательные параметры:
name, description, start_date, end_date

Пример JSON-данных для использования:
{
    "name": "Опрос удовлетворенности EDIT",
    "description": "Общие вопросы"
}

2.3 /api/quiz/<id>/edit DELETE:
Удаляет опрос с номером <id>
Обязательные параметры: нет
Необязательные параметры: нет


Вопросы:
3 /api/question/new POST:
Создаёт вопрос. Обязательные параметры:
quiz_id, type, description.
Необязательные параметры:
answers_choise

Пример JSON-данных для использования:
{
    "quiz_id": 5,
    "type": "choise",
    "description": "Сколько часов в день вы спите?"
}

3.1 /api/question/<id>/edit GET:
Возвращает опрос с номером <id>, с полной информацией
Обязательные параметры: нет
Необязательные параметры: нет

3.2 /api/question/<id>/edit PATCH/PUT:
Обновляет и возвращает вопрос с номером <id>, 
без списка относящихся к нему вопросов.
Обязательные параметры: нет.
Необязательные параметры:
quiz_id, type, description, answers_choise

Пример JSON-данных для использования:
{
	"type": "text",
    "description": "Расскажите, как прошло лето"
}

3.3 /api/question/<id>/edit DELETE:
Удаляет вопрос с номером <id>
Обязательные параметры: нет
Необязательные параметры: нет


---------------------Для участника:-------------------
1. /api/quiz/getallactive GET:
Получить список активных в текущее время опросов.
Обязательные параметры: нет
Необязательные параметры: нет

2. /api/quiz/<id>/ GET:
Получить опрос вместе с входящими в него вопросами.
Обязательные параметры: нет
Необязательные параметры: нет

3. /api/report/<id> GET:
Получает отчёт об прохождении опроса с номером <id>
Обязательные параметры: нет
Необязательные параметры: нет

4. /api/quiz/<id>/report POST:
Прохождение опроса с номером id, генерирует отчёт
report.
Пример JSON-данных для использования:
{
    "quiz_id": 5,
    "user_id": 123,
    "answers": [
        {
            "question_id": 3,
            "value": 2
        },
        {
            "question_id": 4,
            "value": "Примерно 7:30"
        },
        {
            "question_id": 5,
            "value": "Вопросов нет"
        }
    ]
}
